Important HTML tags

This page is designed as a quick reference for some of the common tags covered in Learn Enough HTML to Be Dangerous. In the process of making it, we'll learn how to make HTML tables via table and related tags.

The tables below don't include all HTML tags, but they do list many of the most important ones.

Block Elements

Tag Name Purpose
h1–h6 headings include a heading (levels 1–6)
p paragraph include a paragraph of text
table table include a table
tr table row include a row of data
th table header make a table header
td table data include a table data cell
div division a semantic element, useful for styling
header a header a semantic element, useful for styling
ol ordered list create an ordered list
ul unordered list create an unordered list
li list creates a list item
blockquote block quotation show formatted quotation
br break enter line break

Inline Elements

Tag Name Purpose Example Result
em emphasized make emphasized <em>technical sophistication</em> technical sophistication
strong strong make strong <strong>light</strong> light
a anchor make link <a href="">link<a> link
img image add image <img src="images/kitten.jpg" alt="A Cute Kitten">
code code render code in mono-font <code>puts "Avalon"</code> puts "Avalon"
span span a semantic grouping element, useful for styling <span>some text<span> some text

Document Tags

Tag Name Purpose
html HTML Tells the document this is an HTML document
head Head A container for all the head elements
body Body Defines the documents body
title Title Defines a title in the browser's toolbar
meta Metadata Metadata is data about data
